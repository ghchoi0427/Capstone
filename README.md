# Capstone [창의적 종합설계 경진대회](http://e2festa.kr/ko/capstone)

<h2>Server side</h2>
<h3>🧾 아두이노의 센서값을 전송받아 csv 형식으로 저장한다.</h3>
<h3>📈 저장된 데이터를 그래프로 확인한다.</h3>
<h3>📤 지정된 시간(06시,11시,16시)에 기상 데이터를 전송한다.</h3>

<h2>Client side</h2>
<h3>📡 센서값을 전송한다.</h3>
<h3>📪 기상 데이터 값을 받는다.</h3>
